## API Key 安全

不要将真实 API Key 写进源码或提交到 GitHub。

各章节会提供 `.env.example` 配置模板。将它复制为 `.env` 后，再在本地填写自己的密钥。仓库根目录的 `.gitignore` 会统一忽略 `.env` 文件。

如果密钥曾经出现在公开仓库、截图、录屏或日志中，请立即撤销并重新创建。
