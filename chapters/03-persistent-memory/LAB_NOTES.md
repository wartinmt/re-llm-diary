# 第 03 章实验记录

## 实验 A：跨启动恢复

1. 告诉模型：“我的代号是海盐。”
2. 输入 `/history`，记录消息数量。
3. 输入 `/exit`。
4. 重新运行程序。
5. 再问：“我的代号是什么？”

记录：

- 退出前消息数量：________
- 重启时恢复消息数量：________
- 模型是否回答“海盐”：________

## 实验 B：确认记忆来自文件

1. 输入 `/where`，找到 `conversation.json`。
2. 退出程序。
3. 将文件临时改名为 `conversation.backup.json`。
4. 重新运行并追问代号。
5. 退出，把文件名恢复。
6. 再次运行并追问。

一句话解释结果：

> 模型跨启动记得过去，是因为 ________________________________________________。
